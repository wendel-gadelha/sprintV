Justificativa das Cores usadas:
 Escolha de Cores: Identidade, Experiência, Apelo

O azul escuro é a cor predominante no cabeçalho da nossa plataforma. Ele é frequentemente associado à confiança, segurança e profissionalismo. Essas são qualidades essenciais para uma plataforma de e-commerce, pois queremos que nossos clientes se sintam seguros e confiantes ao realizar suas compras.

O laranja é usado como cor de destaque, especialmente em banners promocionais. Esta cor vibrante transmite energia, entusiasmo e urgência, incentivando os clientes a aproveitar as promoções e ofertas especiais. O laranja também cria um contraste visual forte com o azul escuro, atraindo a atenção dos visitantes para as áreas mais importantes do site.

O branco é utilizado como cor de fundo para criar um layout limpo e organizado. Ele ajuda a destacar os elementos visuais e textuais, facilitando a leitura e a navegação. O uso do branco também transmite uma sensação de espaço e simplicidade, contribuindo para uma experiência de usuário agradável e intuitiva.

Banco de Dados:
Para garantir que a InfintyShop funcione perfeitamente, utilizamos um modelo de banco de dados relacional. Esse modelo nos permite definir claramente as entidades e os relacionamentos entre elas, facilitando a gestão e a consulta dos dados.

Modelagem de Dados: Diagrama Entidade-Relacionamento (DER)
Este é o Diagrama Entidade-Relacionamento (DER) que usamos para modelar as principais entidades do sistema. No DER, definimos as entidades Usuários, Produtos, Pedidos e Fornecedores, e mostramos como elas se relacionam entre si.

Criação de Tabelas na 3FN
As tabelas do banco de dados foram criadas seguindo a Terceira Forma Normal (3FN) para garantir a eliminação de redundâncias e melhorar a integridade dos dados. Vamos ver um exemplo de criação de tabelas com informações de teste.
